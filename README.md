# Protección de Datos y Ciberseguridad

Proyecto web educativo para promover la importancia de la protección de datos y la ciberseguridad. Incluye información clave, principios fundamentales y un test interactivo para que los usuarios evalúen sus conocimientos.

## CaracterÃ­sticas

- **Sección Hero** con mensaje destacado y fondo visualmente atractivo.
- **Explicación sobre la importancia de proteger los datos personales**.
- **Tarjetas de principios clave:** consentimiento, seguridad y responsabilidad Ética.
- **Test interactivo de ciberseguridad** con retroalimentación dinámica.
- **Formulario de contacto** para dudas o consultas.
- **Diseño completamente responsivo** y moderno.
- **Navegación fluida y anclajes ajustados a navbar fija.**

## Tecnologí­as usadas

- HTML5
- CSS3 (personalizado y utilidades Bootstrap 5)
- Bootstrap Icons
- JavaScript (efectos dinámicos y lógica del quiz)

## Integración y uso de AOS Animations

Para mejorar la experiencia visual, se ha integrado la librería [AOS](https://michalsnik.github.io/aos/) que permite animar elementos al hacer scroll.
